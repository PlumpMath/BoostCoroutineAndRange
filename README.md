Build instructions

## Boost Build

Enter the Boost Build directory and execute the following:

$ b2

(You may need to use link=static)

## CMake

Enter the directory ./Coroutine/CMake and execute the following:

$ cmake CMakeLists.txt -b ./
$ cmake --build ./
